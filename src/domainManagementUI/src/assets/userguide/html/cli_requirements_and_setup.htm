CLI Requirements and Setup

 

Requirements

 
The following is a list of prerequisites to run the command line app. You may have to reach out to a Domain Manager Developer for some of these requirements:
 
  • Python 3.8
  • A virtual environment
  • We recommend using the built-in python tool: Venv
  • python3 -m venv.venv
  • Domain Manager API URL
  • Your DM user's API key
 

Setup

 
1. Install Option 1- Local Development
  • Clone the repo to your local environment
  • Git clone https://github.com/cisagov/domain-manager-cli.git
    ii. Install pip install .
    iii. Run "dmcli"
    2. Install Option 2- General Usage
    a. Install directly from repo
    i. pip install git+https://github.com/cisagov/domain-manager-cli.git
    ii. Run "dmcli"
    3. Generate and copy your user API key
  • Login to Domain Manager
     
    Figure: DM login
     
    4. Click on the cog on the top right and then click My Profile
     
Figure: My Profile
 
5. Click on the Options button and then click Generate API key
 
Figure: Generate API Key
 
5. Copy your API key
 
6. Run "dmcli" configuration command
a. dmcli configure
b. When prompted, enter the Domain Manager URL and your API key
 
If you see the returned text below, you're ready to go!
 
Figure: command: dmcli
